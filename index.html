Front-End

Conceitos básicos para iniciar na carreira.

HTML

HTML é a linguagem que cria a estrutura de páginas web, permitindo adicionar textos, imagens e links. Ele organiza o conteúdo, mas não define estilos ou comportamentos.

CSS

CSS deixa nosso site bonito e organizado, permitindo mudar cores, fontes, tamanhos e posicionamento dos elementos. Ele define a aparência de cada parte do site.

Saiba Mais

JAVASCRIPT

JavaScript dá vida ao site, permitindo interatividade, animações e manipulação do conteúdo em tempo real. Com ele, é possível responder a cliques, enviar formulários e atualizar informações sem recarregar a página.
